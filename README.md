# Agentic AI for Finance Transaction Approval

This project is inspired by Microsoft Discovery Hour:
"Transform Finance with Agentic AI in Action".

## Description
A simple Agentic AI system that autonomously evaluates finance transactions
and decides whether to approve, flag, or escalate them with human oversight.

## Concepts Used
- Agentic AI
- Autonomous decision-making
- Human-in-the-loop
- Finance automation

## How to Run
1. Install Python 3
2. Run: python agentic_ai.py
